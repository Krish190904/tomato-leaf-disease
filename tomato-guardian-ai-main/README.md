# FarmGuard AI — Tomato Leaf Disease Detection

AI-powered tomato leaf disease detection system using multimodal deep learning.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or bun

### Installation

```bash
cd tomato-guardian-ai-main
npm install
```

### Development

```bash
npm run dev
```

The app will start on `http://localhost:8080`.

### Build

```bash
npm run build
npm run preview
```

### Testing

```bash
npm test
```

## Tech Stack

- **Frontend:** React 18, TypeScript, Tailwind CSS, shadcn/ui
- **Build Tool:** Vite
- **State Management:** TanStack Query
- **Routing:** React Router v6
- **Backend:** FastAPI (Python)
